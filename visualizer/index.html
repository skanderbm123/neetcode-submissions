Workbench

Manifest —

This page lists every folder under Data Structures & Algorithms that contains submission-*.java files. Run npm run workbench from the repo root to rescan, compile, and run harness tests against your latest submission per folder. Add a new problem folder in the registry when you want Java verification: tools/problem-registry.json plus a harness in tools/harnesses.mjs. When Java verify fails, each problem viz page shows suggested fixes (from compiler output + a small hint catalog). Use npm run workbench:serve for one-click Apply on safe syntax patches, or npm run apply-fix from a terminal.
